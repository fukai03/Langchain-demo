# Langchain Demo
> 学习langchain的简易demo

## 技术栈
- Vite + React18 + ts
- nodejs + express

## 项目结构
- 主页面： `src/App.tsx`
- 服务端文件夹：`server/`

## 项目安装 & 启动

```bash
# 安装前端依赖
yarn
# 安装服务端依赖
cd server && yarn
```

```bash
# 启动前端
yarn dev
# 启动服务端
cd server && node index.js
```

- 前端端口： `localhost:5173`
- 后端端口： `localhost:5003`
